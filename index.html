Entenda o que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas na web.

As tags HTML são os elementos fundamentais que compõem uma página web. Elas são usadas para estruturar o conteúdo e definir a apresentação visual.

Tags Comuns

  • <h1> a <h6>: Cabeçalhos de diferentes níveis
  • <p>: Parágrafo
  • <a>: Serve para colocar links no site, como por exemplo esse: Clique aqui
  • <div>: Serve para agrupar elementos no site
  • <hr>: serve para colocar uma linha horizontal no site
  • <span>: serve para aplicar estilos a partes específicas do conteúdo do site
  • <strong>: serve para colocar o texto em negrito no site
  • <img>: Serve para por imagens no site:
